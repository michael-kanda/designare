Häufige Fragen

Wer ist Michael Kanda?
Michael Kanda ist Fullstack-Entwickler, technischer SEO/GEO-Spezialist und KI-Integrator aus Wien. Er hat über 15 Jahre WordPress-Erfahrung und arbeitet bei der SEO/GEO-Agentur maxonline Marketing.
Was macht Michael Kanda beruflich?
Er entwickelt unter anderem WordPress-Seiten und eigene Plugins, optimiert Performance und Ladezeiten bestehender WordPress-Installationen, setzt Schema.org- und GEO-Strategien um und baut KI-Integrationen in Form von serverlosen Edge-Functions, RAG-Systemen und MCP-Servern.
Was ist ein Web-Handwerker?
Ein Web-Handwerker schreibt Code selbst – ohne PageBuilder, ohne Plugin-Overload, ohne aufgeblähtes Theme. WordPress wird auf das reduziert, was es sein soll: ein schlankes CMS mit sauberer Struktur, das in unter einer Sekunde lädt.
Wo arbeitet Michael Kanda?
100 % remote bei maxonline Marketing, einer SEO/GEO-Agentur mit Sitz in Niederösterreich. Michael selbst lebt und arbeitet von Wien aus.
Welche Tools hat Michael Kanda entwickelt?
Unter anderem die KI-Assistentin Evita (RAG mit Upstash Vector, Redis und Google Gemini), das WordPress-Plugin SEO Wunderkiste, das SEO-Dashboard DataPeak, den designare MCP Server, das Wetter-Script für Google Ads, die LLM-Bulk-Content-Demo Silas und das Plugin CSV Importer PRO.
Welche Technologien beherrscht Michael Kanda?
WordPress Core-Entwicklung, Custom Plugins, PHP, JavaScript, Serverless Edge Functions, Retrieval-Augmented Generation, Vector Databases, Schema.org / JSON-LD, semantisches HTML, Core Web Vitals Tuning, Google Ads Automation und das Model Context Protocol.
Was ist Evita?
Evita ist die KI-Assistentin auf designare.at. Sie basiert auf einem RAG-System mit Upstash Vector, Serverless Redis und Google Gemini, greift in Echtzeit auf Website-Daten und offizielle RSS-Feeds zu und liefert Fakten statt Halluzinationen. Zu ihren Features zählt der Website-Roast, der fremde Seiten nach österreichischem Notensystem (1–5) bewertet.
Wie können LLMs auf designare.at zugreifen?
designare.at betreibt einen eigenen MCP Server unter https://designare.at/api/mcp (Streamable HTTP, JSON-RPC). Registriert als at.designare/knowledge im offiziellen MCP Registry (DNS-verifiziert). Tools: search_knowledge (semantische Suche mit Gemini Embedding und Upstash Vector) und get_services (strukturiertes JSON mit Services und Spezialisierungen). Nächtliche Sync um 3:30 Uhr.
Wie kann man Michael Kanda kontaktieren?
Über das Kontaktformular auf designare.at oder seinen GitHub-Account unter github.com/michael-kanda.
Wofür steht designare.at?
designare.at ist die persönliche Seite von Michael Kanda. Sie dient als Freiraum für Code-Experimente, eigene Tools und technische Deep-Dives zu WordPress, KI-Integration, SEO und GEO – bewusst nicht als Agentur- oder Verkaufsseite.
